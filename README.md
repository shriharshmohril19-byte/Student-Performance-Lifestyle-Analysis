# Student-Performance-Lifestyle-Analysis
An end-to-end data analysis project using SQL and Power BI to explore how student lifestyle factors such as study hours, sleep, attendance, and social time influence academic performance. The project includes data cleaning, exploratory analysis, advanced SQL techniques, and a Power BI dashboard to visualize insights.

Project Overview:

This project analyzes how student lifestyle habits influence academic performance using SQL.

The dataset contains information about:
Student demographics, 
Study hours, 
Sleep hours, 
Social time, 
Attendance, 
GPA / CGPA

The goal is to identify patterns such as:
Which majors perform the best, 
Whether more study hours actually improve grades, 
How sleep impacts academic performance, 
The differences between top and bottom students, etc.

Key Insights

1. Students studying 3–5 hours/day had the highest CGPA on average.

2. Very high study hours showed diminishing returns beyond 7 hours/day.

3. Students sleeping 7–8 hours performed best academically.

4. High attendance strongly correlates with higher CGPA.

5. Top students tend to balance study, sleep, and social time better.
